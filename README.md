# JSONType

Library and CLI utilite for more compact way of showing

```sh
# Install
go install github.com/4nd3r5on/jsontype/cmd/jsontype@latest

# Usage
jsontype -file ./parseme.json
jsontype -file "./parseme1.json, ./parseme2.json, ./parseme3.json"
jsontype -file ./parseme.json -log-level debug
```
